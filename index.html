🌐 Сайт компании Kreuztaler Werkstatt GmbH
📌 Описание проекта
Разработан современный, адаптивный веб-сайт для автосервиса с фокусом на:
  • удобство клиента,
  • онлайн-запись на услуги,
  • эффективное управление заявками через административную панель,
  • интеграцию мгновенных сообщений в Telegram о новых записях клиентов.
Пример уведомления в Telegram:
Новая заявка!  
Имя: Иван Иванов  
E-Mail: ivan@example.com  
Телефон: +7 999 123-45-67  
Услуга: Замена масла  
Дата: 2025-06-18  
Время: 11:00  
Сообщение: Дополнительно проверьте, пожалуйста, работу кондиционера

🖥️ Рекламный раздел

  • Управление заявками

📌 Project Description
Developed a modern, responsive website for a car service center focused on:
  • customer convenience,
  • online booking for services,
  • efficient request management via an admin panel,
  • integration of instant Telegram notifications for new client bookings.
Example Telegram notification:
New request!  
Name: Ivan Ivanov  
E-Mail: ivan@example.com  
Phone: +1 555 123-4567  
Service: Oil change  
Date: 2025-06-18  
Time: 11:00  
Message: Please also check the air conditioner
🔧 Key Features
🖥️ Promo Section
  • Visually appealing homepage
  • Detailed service descriptions
  • Company advantages
  • Client testimonials
  • Photo gallery
📅 Online Booking
  • Convenient booking form for repairs or maintenance
  • Service, date, and time selection
  • Contact information input
  • Confirmation and notifications
👤 Admin Dashboard
  • View booking history
  • Send email reminders to clients
  • Ability to cancel or reschedule bookings
🛠️ Admin Panel
  • Manage requests
  • Manage services
  • Specialist workload control
  • Send reminders to clients
📱 Responsive Design
  • Proper display on: desktops, tablets, smartphones
⚙️ Technologies Used
Frontend: HTML5, CSS3, JavaScript (ES6+), React.js, Bootstrap / Material-UI
Backend: Python (Django or Flask)
Telegram notifications: Integration via Telegram bot
✅ Benefits for Clients and Business
  • Online booking without phone calls
  • Email reminders reduce no-shows
  • Optimized staff workload
  • Improved service quality
  • Increased competitiveness

A digital tool that helps:
attract new clients,
